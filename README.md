# Aidhil Prima Abdiguna — Portfolio

> Website portfolio pribadi — React + Vite + Tailwind CSS v4

Portfolio retro/pixel dengan animasi React Bits, Three.js, dan sistem gradien 3 warna (gold, teal, magenta).

## Tech Stack

![React](https://img.shields.io/badge/React-18-61DAFB?logo=react&logoColor=white)
![Vite](https://img.shields.io/badge/Vite-8-646CFF?logo=vite&logoColor=white)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS_v4-06B6D4?logo=tailwindcss&logoColor=white)
![Three.js](https://img.shields.io/badge/Three.js-r152-000000?logo=three.js&logoColor=white)
![GSAP](https://img.shields.io/badge/GSAP-3-88CE02?logo=greensock&logoColor=white)

- **Framework:** React 18
- **Bundler:** Vite
- **Styling:** Tailwind CSS v4
- **Font:** NK57 Monospace (local .otf)
- **Icons:** Font Awesome 6.5.0 (CDN)
- **3D:** Three.js (PixelBlast background)
- **Animasi:** GSAP + SplitText (TextType)
